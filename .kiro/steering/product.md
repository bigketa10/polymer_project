# Product: PolymerLingo

PolymerLingo is a gamified polymer chemistry learning platform for students. It delivers bite-sized interactive lessons with XP rewards, streaks, and a leaderboard to make learning polymer science engaging.

## Core User Roles

- **Students** — complete lessons, earn XP, maintain streaks, view progress
- **Teachers** — manage modules, lessons, questions, glossary, and view class analytics

## Key Features

- Multiple question types: multiple choice (MCQ), drag & drop, fill-in-the-blank
- Module/lesson hierarchy with ordering and difficulty levels
- Student leaderboard and progress tracking
- Teacher dashboard with class stats, student reports, and content management
- Image uploads per question (stored in Convex storage)
- Glossary management
